# Vehicle Manager -- Help

Welcome to the Vehicle Manager documentation. This application provides digital damage documentation and fleet vehicle management.

## Features

| Area | Description |
|---|---|
| **Vehicle Management** | Create, edit, search, and deactivate vehicles |
| **Vehicle Types** | Manage vehicle types with custom view images |
| **Damage Marking** | Interactively mark damage on a 4-side vehicle view |
| **Damage Repair** | Mark repaired damages as resolved |
| **Export & Reports** | PNG export, damage reports, print view |
| **QR Codes** | Generate QR codes for checklist forms |
| **Driver Management** | Create and manage drivers |
| **Checklists** | Public handover form and admin overview |

## Access

The application is protected by login. Registration is restricted to company email addresses. After registration, the email address must be verified using a 6-digit code.

## Navigation

The main navigation is located at the top of the page:

- **Fahrzeuge** -- Vehicle list (home page)
- **Fahrzeugtypen** -- Vehicle types with custom images
- **Fahrer** -- Driver management
- **Checklisten** -- View checklist submissions
- **Hilfe** -- This documentation
