# Vehicle Types

Vehicle types allow you to upload custom view images for different vehicle models. These images are displayed as backgrounds on the damage marking canvas.

## Create a Type

1. Navigate to **Fahrzeugtypen** (Vehicle Types) in the main navigation.
2. Click **Fahrzeugtyp hinzufugen** (Add vehicle type).
3. Enter a **name** (e.g., "Mercedes Sprinter", "MAN TGE").
4. Click **Speichern** (Save).

## Upload Images

After creating a type, you can upload an image for each of the four views:

- **Front** -- Front view
- **Rear** -- Rear view
- **Left** -- Left side
- **Right** -- Right side

### How to upload:

1. Click **Hochladen** (Upload) under the desired view.
2. Select an image file (JPEG, PNG, WebP, or SVG).
3. The image is uploaded immediately and displayed as a preview.
4. To replace an image, click **Hochladen** again.

> Tip: Use transparent PNG or SVG files for the best results on the damage marking canvas.

## Edit a Type

Click **Bearbeiten** (Edit) next to the type name to change the name.

## Delete a Type

Click **Loschen** (Delete). Vehicles using this type will lose their type assignment (the vehicles themselves are preserved).

## Assigning to Vehicles

When creating or editing a vehicle, you can select a vehicle type from the dropdown. The uploaded images will then be used as the background in the damage view.
