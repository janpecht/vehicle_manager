# Fahrzeugmanager -- Hilfe

Willkommen in der Dokumentation des Fahrzeugmanagers. Diese Anwendung dient der digitalen Schadensdokumentation und Fahrzeugverwaltung Ihrer Flotte.

## Funktionsumfang

| Bereich | Beschreibung |
|---|---|
| **Fahrzeugverwaltung** | Fahrzeuge anlegen, bearbeiten, suchen und deaktivieren |
| **Fahrzeugtypen** | Fahrzeugtypen mit eigenen Ansichtsbildern verwalten |
| **Schadenmarkierung** | Schaden interaktiv auf einer 4-Seiten-Ansicht markieren |
| **Schadenreparatur** | Reparierte Schaden als erledigt kennzeichnen |
| **Export & Berichte** | PNG-Export, Schadenberichte, Druckansicht |
| **QR-Codes** | QR-Codes fur Checklisten-Formulare generieren |
| **Fahrerverwaltung** | Fahrer anlegen und verwalten |
| **Checklisten** | Offentliches Ubergabe-Formular und Admin-Ubersicht |

## Zugang

Die Anwendung ist durch ein Login geschutzt. Registrierung ist auf E-Mail-Adressen der Firma beschrankt. Nach der Registrierung muss die E-Mail-Adresse uber einen 6-stelligen Code verifiziert werden.

## Navigation

Die Hauptnavigation befindet sich im oberen Bereich der Seite:

- **Fahrzeuge** -- Fahrzeugliste (Startseite)
- **Fahrzeugtypen** -- Typen mit eigenen Bildern verwalten
- **Fahrer** -- Fahrerverwaltung
- **Checklisten** -- Checklisten-Einreichungen einsehen
- **Hilfe** -- Diese Dokumentation
