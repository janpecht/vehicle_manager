# Schadenreparatur

Reparierte Schaden konnen als erledigt markiert werden, ohne sie zu loschen. So bleibt die Schadenshistorie erhalten.

## Schaden als repariert markieren

1. Offnen Sie die Fahrzeug-Detailseite.
2. Klicken Sie auf die Schadensmarkierung auf der Leinwand.
3. Im Detail-Popup klicken Sie auf **Als repariert markieren**.
4. Bestatigen Sie die Aktion im Bestatigungsdialog.

## Darstellung reparierter Schaden

Reparierte Schaden werden visuell unterschieden:

- **Gestrichelter Umriss** statt durchgezogener Linie
- **Transparente Fullung** statt deckender Farbe
- Im Detail-Popup wird das **Reparaturdatum** angezeigt

## Filter

Mit dem Toggle **Reparierte anzeigen** in der Werkzeugleiste konnen Sie reparierte Schaden ein- und ausblenden. Standardmassig werden nur aktive Schaden angezeigt.

## Schaden loschen

Aktive (nicht reparierte) Schaden konnen uber das Detail-Popup geloscht werden:

1. Klicken Sie auf die Markierung.
2. Klicken Sie auf **Loschen**.
3. Bestatigen Sie die Loschung.

> Hinweis: Das Loschen entfernt den Schaden permanent. Verwenden Sie stattdessen "Als repariert markieren", um die Historie zu erhalten.
