# Authentifizierung

## Registrierung

1. Offnen Sie die **Registrierungsseite** uber den Link auf der Login-Seite.
2. Geben Sie Ihren **Namen**, Ihre **E-Mail-Adresse** und ein **Passwort** ein.
3. Das Passwort muss mindestens 8 Zeichen lang sein und Grossbuchstaben, Kleinbuchstaben und Ziffern enthalten.
4. Nach der Registrierung wird ein **6-stelliger Verifizierungscode** an Ihre E-Mail gesendet.

> Hinweis: Nur E-Mail-Adressen der Firmen-Domain konnen sich registrieren.

## E-Mail-Verifizierung

1. Geben Sie den 6-stelligen Code ein, den Sie per E-Mail erhalten haben.
2. Die Eingabefelder springen automatisch weiter. Sie konnen den Code auch einfugen (Strg+V).
3. Falls der Code nicht angekommen ist, klicken Sie auf **Code erneut senden** (nach 60 Sekunden verfugbar).
4. Nach erfolgreicher Verifizierung werden Sie zum Login weitergeleitet.

## Login

1. Geben Sie Ihre **E-Mail-Adresse** und Ihr **Passwort** ein.
2. Nach erfolgreichem Login gelangen Sie zur Fahrzeugliste.
3. Ihre Sitzung bleibt aktiv und wird automatisch im Hintergrund verlangert.

## Passwort vergessen

1. Klicken Sie auf der Login-Seite auf **Passwort vergessen?**
2. Geben Sie Ihre E-Mail-Adresse ein.
3. Sie erhalten einen **6-stelligen Reset-Code** per E-Mail (gultig fur 15 Minuten).
4. Geben Sie den Code und Ihr neues Passwort auf der Reset-Seite ein.
5. Nach dem Zurucksetzen werden alle bestehenden Sitzungen beendet.

## Abmelden

Klicken Sie oben rechts auf **Abmelden**, um sich auszuloggen.
