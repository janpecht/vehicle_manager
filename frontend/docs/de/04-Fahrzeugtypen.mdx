# Fahrzeugtypen

Fahrzeugtypen ermoglichen es, benutzerdefinierte Ansichtsbilder fur verschiedene Fahrzeugmodelle zu hinterlegen. Diese Bilder werden als Hintergrund auf der Schadensmarkierungs-Leinwand angezeigt.

## Typ anlegen

1. Navigieren Sie zu **Fahrzeugtypen** in der Hauptnavigation.
2. Klicken Sie auf **Fahrzeugtyp hinzufugen**.
3. Vergeben Sie einen **Namen** (z.B. "Mercedes Sprinter", "MAN TGE").
4. Klicken Sie auf **Speichern**.

## Bilder hochladen

Nach dem Anlegen eines Typs konnen Sie fur jede der vier Ansichten ein Bild hochladen:

- **Front** -- Vorderansicht
- **Heck** -- Heckansicht
- **Links** -- Linke Seite
- **Rechts** -- Rechte Seite

### So laden Sie ein Bild hoch:

1. Klicken Sie unter der gewunschten Ansicht auf **Hochladen**.
2. Wahlen Sie eine Bilddatei aus (JPEG, PNG, WebP oder SVG).
3. Das Bild wird sofort hochgeladen und als Vorschau angezeigt.
4. Um ein Bild zu ersetzen, klicken Sie erneut auf **Hochladen**.

> Tipp: Verwenden Sie transparente PNG- oder SVG-Dateien fur die besten Ergebnisse auf der Schadensmarkierungs-Leinwand.

## Typ bearbeiten

Klicken Sie auf **Bearbeiten** neben dem Typnamen, um den Namen zu andern.

## Typ loschen

Klicken Sie auf **Loschen**. Fahrzeuge, die diesen Typ verwenden, verlieren ihre Typzuordnung (die Fahrzeuge selbst bleiben erhalten).

## Zuordnung zu Fahrzeugen

Beim Anlegen oder Bearbeiten eines Fahrzeugs konnen Sie einen Fahrzeugtyp aus dem Dropdown auswahlen. Die hinterlegten Bilder werden dann als Hintergrund in der Schadensansicht verwendet.
